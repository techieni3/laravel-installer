# Laravel Installer

This repo contains opinionated versions of the Laravel Installer.

## Installation

You can install this package using composer:

```bash
composer global require techieni3/laravel-installer
```

Create new Project:

```bash
laravel new example-app
```

## Testing

```bash
composer test
```
